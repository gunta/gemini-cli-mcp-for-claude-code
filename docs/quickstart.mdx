## Installation

Choose your preferred installation method:

<Tabs>
  <Tab title="DXT (Recommended)">
    The easiest way to install is using DXT, Anthropic's official MCP server manager:

    ```bash
    dxt install gemini-cli-mcp-for-claude
    ```

    That's it! DXT will handle all the configuration for you.
  </Tab>
  
  <Tab title="Manual Installation">
    If you prefer manual installation:

    ```bash
    # Clone the repository
    git clone https://github.com/gunta/gemini-cli-mcp-for-claude-code.git
    cd gemini-cli-mcp-for-claude-code

    # Install dependencies
    npm install

    # Build the project
    npm run build
    ```
  </Tab>
</Tabs>

## Configuration

### 1. Get your Gemini API Key

First, you'll need a Google Gemini API key:

<Steps>
  <Step title="Visit Google AI Studio">
    Go to [Google AI Studio](https://makersuite.google.com/app/apikey)
  </Step>
  <Step title="Create API Key">
    Click "Create API Key" and copy it
  </Step>
  <Step title="Save securely">
    Store your API key in a secure location
  </Step>
</Steps>

### 2. Configure Claude Desktop

Add the MCP server to your Claude Desktop configuration:

<CodeGroup>
```json macOS
{
  "mcpServers": {
    "gemini": {
      "command": "node",
      "args": ["/path/to/gemini-cli-mcp-for-claude/dist/index.js"],
      "env": {
        "GEMINI_API_KEY": "your-api-key-here"
      }
    }
  }
}
```

```json Windows
{
  "mcpServers": {
    "gemini": {
      "command": "node",
      "args": ["C:\\path\\to\\gemini-cli-mcp-for-claude\\dist\\index.js"],
      "env": {
        "GEMINI_API_KEY": "your-api-key-here"
      }
    }
  }
}
```

```json Linux
{
  "mcpServers": {
    "gemini": {
      "command": "node",
      "args": ["/home/user/gemini-cli-mcp-for-claude/dist/index.js"],
      "env": {
        "GEMINI_API_KEY": "your-api-key-here"
      }
    }
  }
}
```
</CodeGroup>

<Note>
  The configuration file location varies by platform:
  - **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
  - **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`
  - **Linux**: `~/.config/Claude/claude_desktop_config.json`
</Note>

### 3. Environment Variables

You can configure the following environment variables:

| Variable | Description | Default |
|----------|-------------|---------|
| `GEMINI_API_KEY` | Your Google Gemini API key | Required |
| `GEMINI_CLI_PATH` | Path to Gemini CLI (for full features) | Optional |
| `GEMINI_MODEL` | Default model to use | `gemini-2.0-flash-thinking-exp-1219` |
| `GEMINI_TEMPERATURE` | Default temperature | `0.7` |
| `GEMINI_MAX_TOKENS` | Default max tokens | `8192` |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Claude doesn't recognize the MCP server">
    1. Ensure Claude Desktop is fully closed
    2. Verify your configuration file is valid JSON
    3. Check the path to the MCP server is correct
    4. Restart Claude Desktop
  </Accordion>
  
  <Accordion title="API key errors">
    1. Verify your API key is valid
    2. Check you have the correct permissions
    3. Ensure the key hasn't expired
    4. Try generating a new API key
  </Accordion>
  
  <Accordion title="Generation failures">
    1. Check your internet connection
    2. Verify you haven't exceeded API quotas
    3. Try a simpler prompt first
    4. Check the error message for specific issues
  </Accordion>
</AccordionGroup>
